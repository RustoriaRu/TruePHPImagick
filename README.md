Это расширение PHP для проекта Рустория.

В нём содержится утилитарный функционал расширяющий стандартные возможности PHP.

Установка производится с помощью утилиты phpize, макрос установки находится в sh скрипте build.sh
Для очистки каталога от результатов адаптации phpize следует вызывать скрипт clean.sh
Для проверки функционала вызывается test.sh
